os_queue_doubly_sort_t os_event_descriptor::sort

Sorting function. This function compare 2 processes. The following helper functions are available:

  • os_queue_sort_fifo
  • os_queue_sort_lifo.

Note:
This function is optional. By default the new process will added using the os_queue_sort_fifo algorithm or the os_queue_doubly_process_sort_priority alorithm, depending if priorities are enabled or not.
