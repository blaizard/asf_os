Singly Queues

Singly queues are singly linked list. Singly linked lists contain nodes which have a data field as well as a next field, which points to the next node in the linked list.

Doubly Queues

Doubly queues are doubly linked lists. In a doubly linked list, each node contains, besides the next-node link, a second link field pointing to the previous node in the sequence.

Macro used to define a new singly queue type.

The queue type defined by this macro will inherit from all the following functions (from the os_queue):

  • os_queue_<NAME>_pop alias of os_queue_pop
  • os_queue_<NAME>_head alias of os_queue_head
  • os_queue_<NAME>_insert_after alias of os_queue_insert_after
  • os_queue_<NAME>_insert_first alias of os_queue_insert_first
  • os_queue_<NAME>_add_sort alias of os_queue_add_sort
  • os_queue_<NAME>_remove alias of os_queue_remove

But also from 2 other functions to cast the defined type:

  • os_queue_<NAME>_get_queue (get the os_queue element from the os_<NAME>_queue type)
  • os_queue_<NAME>_from_queue (get the os_<NAME>_queue element from the os_queue type)
Parameters:
NAME Prefix for the name of the queue. The queue type defined will be a structure with the following symbol: os_queue_<NAME>
