Enumeration Type Documentation

enum os_event_status

Status returned by an event.

Enumerator:
OS_EVENT_NONE 

No event has occured.

OS_EVENT_OK_STOP 

An event has been triggered for this process. If another process is sharing the same event, there is no need to test if this event is also valid for it. In other word, this event is only valid for 1 process.

OS_EVENT_OK_CONTINUE 

An event has been triggered for this process. If another process is sharing the same event, we might want to also chech if this event is also valid. In other word, this event might be valid for more than 1 process.
