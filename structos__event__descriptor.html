bool(* os_event_descriptor::sort)(struct os_process *proc1, struct os_process *proc2)

Sorting function. This function compare 2 processes. The following helper functions are available:

  • os_event_sort_fifo
  • os_event_sort_lifo
    Note:
    This function is optional. By default the new process will added using the os_event_sort_fifo algorithm or the os_event_sort_priority alorithm, depending if priorities are enabled or not.
    Parameters:
    proc1The first process used in the comparison
    proc2The second process used in the comparison
    Returns:
    true if proc1 should be placed before proc2, false otherwise.
