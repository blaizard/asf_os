Detailed Description

Type definition for the structure os_queue_doubly_process.

The os_queue_doubly_process is a singly queue which holds a process information. This information is extracted from a queue element using the function os_queue_doubly_process_get. os_queue_doubly_process is a child of os_queue_doubly and therefore inherits from all its functions. For more details, see OS_QUEUE_DOUBLY_DEFINE.
