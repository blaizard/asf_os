#define CONFIG_OS_PROCESS_ENABLE_FIFO   false

Setting this define to true will bring enabled process at the end of the active process list. The last process enabled will be the last called by the scheduler. For example, if you enable process_3 to the active process list.
if CONFIG_OS_PROCESS_ENABLE_FIFO == true. 

 process_1 -> process_2 -> process_3 -> process_1 -> ...

if CONFIG_OS_PROCESS_ENABLE_FIFO == false 

 process_3 -> process_1 -> process_2 -> process_3 -> ...
Note:
Activating this define impacts a bit the CPU and memory ressources when enabling a new process.
