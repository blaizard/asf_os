enum os_task_option
Enumerator:
OS_TASK_DEFAULT 

Default options.

OS_TASK_DISABLE 

Disable the task before its execution.
It can be enable at any time using os_task_enable.

OS_TASK_USE_CUSTOM_STACK 

Use a custom stack for this task. The user must previously allocate memory for os_task::stack. This option is available only if CONFIG_OS_USE_MALLOC is set.
